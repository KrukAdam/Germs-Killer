A mini game made in 48 hours.

## Technologies
Project is created with:
* Unity 2018.4.7f1
* C#

## Features
- Timer and levels
- Health 
- Enemy have speed, size, health depending on the type
- Random time respawn
- Respawn type of enemy depending on the level
